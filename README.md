# Go Beginner Toolkit – HTTP Hello World Server

#### Date 23/02/2025

## Project Overview

This project is part of the Moringa AI Capstone:

### Prompt-Powered Kickstart: Building a Beginner’s Toolkit

It demonstrates how to:

- Install Go

- Set up a project

- Build a simple HTTP server

- Use AI prompts to assist learning

## What This Project Does

This Go program:

- Starts a web server

- Listens on port 8080

Displays:
```
Hello, World from Go!
```
in the browser.

## Tech Stack

- Go (Golang)

- Built-in net/http package

- Git & GitHub

Go was created by Google and is widely used for backend systems.

## Requirements

- Go 1.20 or higher

- Terminal / Command Prompt

- Web Browser

Check installation:
```
go version
```

## How to Run the Project
1. Clone the Repository
```
git clone git@github.com:Bonarerijt/go-beginner-toolkit.git
cd go-beginner-toolkit
```

2. Run the Server
```
go run main.go
```
You should see:
```
Server running at http://localhost:8080
```

3. Open in Browser

Visit:
```
http://localhost:8080
```

Expected Output:
```
Hello, World from Go!
```

## Common Issues

| Issue                   | Solution                      |
| ----------------------- | ----------------------------- |
| `go: command not found` | Install Go & restart terminal |
| Port already in use     | Change port number in code    |
| Module error            | Run `go mod init hello`       |
